# Public Reverse Engineering Index

Status: public-safe overview
Last updated: 2026-06-22

This public index summarizes the reverse-engineering material that is suitable
for normal source collaboration. The private maintainer tree contains much
deeper Ghidra wave notes, proof sidecars, runtime evidence, and local backup
records; those are intentionally not part of the public candidate by default.

## What Is Public Here

- [Quick Reference](quick-reference/_index.md) - compact tables for saves,
  resources, command-line flags, source hierarchy, and script commands.
- [Static Contracts Overview](public-static-contracts.md) - current static
  closure and proof-boundary summary.
- [Save And Options Overview](public-save-options.md) - public-safe `.bes` and
  `defaultoptions.bea` rules for contributors.
- [Assets And Modding Overview](public-assets-and-modding.md) - public-safe
  resource/export/modding boundaries.

## Current Static Boundary

The private Ghidra database has function-quality closure and current-risk
static re-audit closure in the maintainer accounting files. Public docs should
state that bounded static contracts exist, but they must not translate static
closure into runtime gameplay proof, patch behavior proof, rebuild parity, or
no-noticeable-difference parity.

## What Is Not Public Payload

The public candidate excludes private game files, copied executables, saves,
screenshots, frame captures, Ghidra backups, local proof bundles, raw runtime
artifacts, and operator state files. Contributors should use their own legally
obtained game installation and keep proprietary game content out of PRs.

## Maintainer Note

If a deeper proof document should become public, export it as a closed subgraph:
all local markdown links must resolve inside the public candidate, and any
sidecar JSON/TSV must pass the public payload safety checks.
