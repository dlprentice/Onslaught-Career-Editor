## Public Boundary

The public book intentionally avoids maintainer-only proof batons, Ghidra
backup paths, raw runtime evidence, screenshots, copied executable artifacts,
and private game assets. Use [Security And Private Data Reporting](../SECURITY.md)
before discussing anything that might contain private or proprietary material.
