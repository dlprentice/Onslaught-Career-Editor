# WinUI Model Sidecar Texture Visual Smoke - 2026-05-08

## Scope

This note records a focused WinUI Asset Library native UI Automation smoke for the model texture sidecar fallback path.

The fixture is generated by the UI test itself. It contains one loose FBX model export with a texture binding, zero texture catalog rows, and one local `MeshTextures` sidecar PNG. The proof stays public-safe because it does not use or commit private game assets, private screenshots, copied executables, runtime captures, or raw proof JSON.

## Result

Status: PASS.

The WinUI Asset Library now has a deterministic native smoke for a sidecar-only model row:

- The model row selects `sidecar_only_mesh.msh`.
- The model texture summary reports `none are direct catalog rows`.
- The model texture summary reports `Sidecar preview files: 1/1`.
- The primary model texture action is `Preview sidecar texture`.
- Invoking the action shows `Sidecar texture preview: orphan_sidecar.png`.
- The ignored screenshot `asset-library-sidecar-texture.png` shows the maximized WinUI Asset Library with the sidecar preview visible below the wireframe model panel.
