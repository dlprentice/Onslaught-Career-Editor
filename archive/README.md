# Archive

Status: active archive
Last updated: 2026-05-05

This folder holds legacy app surfaces and historical planning snapshots that are retained for reference and private fallback work only.

Current contents:

- `legacy-wpf/` - archived WPF desktop app surface
- `legacy-winui-release/` - archived WinUI portable-bundle helper scripts/docs
- `legacy-python/` - archived Python GUI/CLI app surface
- `electron-workbench/` - archived Electron/React/TypeScript workbench and TypeScript CLI
- `historical-docs/` - deprecated planning notes and superseded roadmap records that have current replacements elsewhere

Shipping rule:

- this archive is non-shipping and is excluded from curated public/community outputs
- WinUI 3 is the active product surface
- AppCore, AppCore.Host, and the C# CLI are active support/reference surfaces, not archived apps

Maintenance rule:

- do not add new product features here unless the user explicitly revives a legacy lane
- if a legacy file is still operationally useful, keep it archived and clearly marked instead of pretending it is current
