## Start Here

For the WinUI Asset Library, first generate a local asset catalog from your own
Battle Engine Aquila install, then load the generated export folder in the app.
The folder should contain `asset_catalog/catalog.json`; the game install folder
itself is not an Asset Library catalog.

Current extraction workflow: [extraction-pipeline.md](extraction-pipeline.md)

Local overlay rules: [../../LOCAL_LAB_OVERLAY.md](../../LOCAL_LAB_OVERLAY.md)

The public repo tracks tools, docs, contracts, and public-safe summaries. It does
not track extracted game assets, copied executables, private proof captures, or
full Ghidra project stores.
