> Index for per-source-file function documentation under `functions/`.

## Overview

This folder contains function mappings grouped by original source file name
(e.g., `Career.cpp/`, `Camera.cpp/`). Each subfolder includes an `_index.md`
that lists functions and links to per-function docs where available.

Some function links intentionally point to **placeholder docs** that mark
future analysis work. These placeholders exist to keep index navigation
consistent and to avoid broken links.
